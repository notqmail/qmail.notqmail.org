NAME
     qmail-clean - clean up the queue directory

SYNOPSIS
     qmail-clean

DESCRIPTION
     qmail-clean reads a cleanup command from descriptor 0,  per-
     forms  the  cleanup, prints the results to descriptor 1, and
     repeats.

SEE ALSO
     qmail-send(8)
