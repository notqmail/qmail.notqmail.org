error_str(3)           C Library Functions           error_str(3)



NAME
     error_str - names for syscall error codes

SYNTAX
     #include <error.h>

     char *error_str(e);

     int e;

DESCRIPTION
     error_str  returns  a  printable  string  describing syscall
     error code e.  Normally e is errno.

SEE ALSO
     error(3)
