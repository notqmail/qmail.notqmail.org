now(3)                 C Library Functions                 now(3)



NAME
     now - get current time, in seconds

SYNTAX
     #include <now.h>

     datetime_sec now();

DESCRIPTION
     now  returns  the  number  of  real-time  seconds  that have
     elapsed since the end of 1969 TAI.

SEE ALSO
     datetime(3), time(3)
