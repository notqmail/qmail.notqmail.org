Run packages

The packages listed here help you run various services such as qmail and djbdns using daemontools. As a minimum, you need to install the functions, runlevelconf, and dt-run packages. This will enable you to run all DJB packages to run under daemontools except qmail. To run qmail under daemontools, install the qmail-run package. If you want to run some non-DJB packages, such as rsync, proftpd, telnetd, under daemontools, install the other-run package.

functions

This package is required by all the others. It provides some shell functions that helps to set up PATH-like environments.
